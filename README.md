# Атомный Конь

## План программного проекта

### Описание проекта

«Атомный Конь» — это однопользовательская логическая игра, реализующая задачу обхода шахматной доски ходом коня. Игрок перемещает фигуру по правилам шахматного коня (две клетки в одном направлении и одну — в перпендикулярном) с целью посетить каждую клетку игрового поля ровно один раз. Приложение реализуется в виде адаптивного веб-интерфейса с возможностью выбора размера доски и отслеживания прогресса игры.

---

## Модули программного проекта

* **Интерфейс пользователя:** Клиентская часть обеспечивает взаимодействие пользователя с игровым процессом. Интерфейс должен отображать игровое поле динамического размера, текущую позицию коня, историю ходов и таймер. Пользователь получает возможность выбирать размер доски, выполнять ходы кликом по клетке и отменять последнее действие. Интерфейс должен быть интуитивно понятным и адаптивным для различных устройств.

* **Модуль управления игровой логикой:** Данный модуль реализует основную бизнес-логику приложения. Он отвечает за генерацию доски заданного размера, хранение состояния клеток, проверку корректности ходов коня, запрет повторного посещения клеток и определение завершения игры. Модуль также обеспечивает корректную обработку отмены хода и обновление состояния системы после каждого действия пользователя.

* **Модуль истории ходов и таймера:** Компонент отвечает за сохранение последовательности ходов игрока и отображение времени прохождения партии. История должна позволять восстановление предыдущего состояния при отмене действия. Таймер запускается при начале игры, останавливается при её завершении и сбрасывается при создании новой партии.

* **Модуль сборки и развертывания:** Обеспечивает подготовку проекта к запуску и публикации. Включает настройку сборки Angular-приложения, тестирование бизнес-логики и контейнеризацию с использованием Docker и nginx для развертывания production-версии.

---

## Используемые технологии

Angular 18, TypeScript, Lucide Icons, ESLint, Prettier, Jasmine, Docker, nginx.

---

## Ожидаемый результат

В результате разработки будет создано адаптивное веб-приложение, реализующее задачу обхода шахматной доски конём, с возможностью выбора сложности, ведением истории ходов и отслеживанием времени прохождения. Приложение должно быть логически структурировано, протестировано и готово к развертыванию в контейнеризированной среде.
